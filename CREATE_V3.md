# Bot V3 重构计划

## 当前问题
1. bytes处理不一致导致翻页失败
2. 缺少智能引导流程  
3. 代码复杂度高，难以维护

## 解决方案
创建bot_v3_simple.py - 简洁版本，包含：

### 核心功能
1. ✅ 基础搜索（/search, /text等）
2. ✅ 缓存和自动翻页
3. ✅ bytes统一处理
4. ✅ 简化的AI引导：
   - 用户输入 → AI生成3个建议按钮
   - 用户点击 → 执行搜索
   - 结果底部有"返回重选"按钮

### 代码结构
- 单文件，清晰分段
- 工具函数集中在顶部
- 所有bytes用hex存储
- 完整错误处理
